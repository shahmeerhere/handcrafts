# Morva

**Morva** is a fully functional, responsive online shoe store built with **Next.js** and **Tailwind CSS**. It provides users with a seamless shopping experience, allowing them to browse products, filter by category or price, and add items to their cart, all with a modern, responsive design.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/shahmeerhere/shoestore.git
```

2. Navigate into the project folder:

```bash
cd morva
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) to explore the store.

## Usage

1. Browse the shoe catalog.
2. Use filters and sorting to find the perfect product.
3. Add items to your cart and view the cart summary.
4. (Optional) Integrate payment and database for full e-commerce functionality.
